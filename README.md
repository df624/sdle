# SDLE

## How to run
- Run Proxy
-> cd src
-> python proxy.py

- Run Servers
-> cd src
-> python worker.py <worker id>
(repeat 5 times in 5 terminals to create 5 servers)

- Run Client
-> cd src
-> python Client.py
